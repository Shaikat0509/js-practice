* [11, 20, 34, 45, 53, 69, 75, 80, 99, 105]  এই অ্যারে (Array) এর  ম্যাপ map() দিয়ে দেখতে হবে সেটা ৫ দিয়ে ভাগ যায় কিনা। দেখতে হবে এইভাবে 
```sh

    '11 -> NO', 
    '20 -> YES', 
    ..........
}
```
